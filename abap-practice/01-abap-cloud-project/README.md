# Exercise 1: Create an ABAP Cloud Project and Investigate ABAP Coding

## 목적
- Eclipse의 ABAP 개발 환경과 기본 탐색 기능을 직접 사용해 본다.

## 한 일
- ABAP perspective에서 ABAP Cloud Project를 생성했다.
- `/DMO/FLIGHT`를 Favorite Packages에 추가했다.
- `/DMO/CL_FLIGHT_LEGACY`를 열어 Properties와 Project Explorer 연동을 확인했다.
- 소스 검색, 구현 이동, code element 정보와 ABAP Language Help를 사용했다.

## 한 줄 정리
- ABAP Development Tools에서 개발 객체를 찾고 관련 정보로 이동하는 기본 흐름을 익혔다.
